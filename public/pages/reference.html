⛔ NEVER laser: PVC, vinyl, or any chlorine-containing material. Releases toxic gas. Check material before use.
Pre-Job Safety Checklist
  • Ventilation running / window open
  • Material confirmed — no PVC or vinyl
  • Auto-focus run (or manual focus set)
  • Red cross alignment verified on workpiece
  • Lid closed / enclosure secured
  • Fire extinguisher within reach
  • Do not leave machine unattended during run
Operation Mode Guide
ModePath typeTypical useSpeed

Decision: vector line art → Score. Filled area / photo → Engrave. Through-cut → Cut.

AI → GIMP → Inkscape → Laser Workflow
  1. AI — generate or source image
  2. GIMP — adjust contrast, greyscale, clean noise → export PNG
  3. Inkscape — Path › Trace Bitmap → simplify → export SVG
  4. XCS / LightBurn — import SVG, set parameters, run

File naming: ProjectName_Stage_Version_Date.ext
e.g. FlowerDesign_Vector_V2_230424.svg

Inlay Painting Steps
  1. Engrave design to desired depth
  2. Apply acrylic paint into recessed areas
  3. Let dry fully
  4. Sand flush with surface (fine grit)
  5. Repeat for additional colours with masking
  6. Apply clear coat to protect inlay
Material Test Procedure
  1. Use XCS Material Test Array feature (or manually set up a grid)
  2. Set a range of power (e.g. 30–80%) and speed (e.g. 200–400 mm/sec)
  3. Run array on scrap of target material
  4. Identify best result — note power, speed, LPI, passes
  5. Record in Settings database with ★ starred
